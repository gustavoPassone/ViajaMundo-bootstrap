## 📖 Sobre
Portal turístico focado em **visual impactante**. Este projeto explora componentes de mídia do Bootstrap 5, como Carrosséis automáticos e Grids de Cards com posicionamento avançado de elementos.

## 🌟 Funcionalidades
- **Hero Slider:** Carrossel de imagens com legendas e controles.
- **Cards de Pacotes:** Layout complexo com imagem, badges de desconto (overlay) e avaliação.
- **Barra de Busca:** Formulário inline com inputs de texto e data.
- **Responsividade:** Adaptação total para mobile e tablet.

## 🛠 Tecnologias
- HTML5
- Bootstrap 5
- Bootstrap Icons
